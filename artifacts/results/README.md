# Experimental Results

This directory stores the outputs generated during model training and evaluation.

Typical contents include:

- Training and validation metrics
- Evaluation reports
- Confusion matrices
- ROC and Precision–Recall curves
- Explainability outputs
- Additional experiment artifacts

These files are generated during execution and are not required to use the repository. Running the training and evaluation pipeline will recreate the appropriate outputs.

For the complete execution workflow, see:

- `notebooks/00_getting_started.ipynb`
- `README.md`
