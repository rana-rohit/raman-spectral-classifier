# Archive

This directory contains archived development notebooks. 

These notebooks document exploratory analyses, debugging investigations, preprocessing validation, and development experiments performed during the construction of the Raman spectroscopy bacterial classification framework.

They are retained for transparency and historical reproducibility.

**Note:** The analyses documented here supported repository development but are **not part of the finalized methodology**.

The finalized methodology is documented exclusively in:
- `00_getting_started.ipynb`
- `01_dataset_exploration.ipynb`
- `02_train_and_evaluate.ipynb`
- `03_explainability.ipynb`
